Read the grid, plan the week

  • CN + C++ column → TCP deep dive, epoll, subnetting, vtables, memory layout — this is Arista prep, and half of Citadel's C++ screen.
  • Concurrency column → multithreading in C++, semaphore vs mutex, locks internals, CS:APP ch 12 — Rubrik's two system-coding rounds live here.
  • Probability column → the whole Probability & Puzzles subject — D. E. Shaw aptitude and Citadel screens.
  • Practical column → can't cram it; practice narrating, testing, and reading unfamiliar code (Stripe bug-squash, Intuit craft demo).
